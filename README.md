## ISHÉ Clothing

This is a Woman clothing E-commerce website built using HTML and CSS. Only frontend design for now.

## Features

- Responsive layout
- Scrollable Product gallery (both ways)
- Navigation menu
- Attractive section headers using logos
- Footer with social links

## Technologies Used

- HTML
- CSS
- JavaScript

## Folder Structure

```
index.html
style.css
script.js
images/
```
